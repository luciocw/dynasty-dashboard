## English

### What is it?
A lightweight, single-file web dashboard to view all your Sleeper fantasy football leagues, rosters, draft picks, and standings at a glance. Built for dynasty managers who play in multiple leagues.

### Features
- **Multi-league view** - See all leagues in cards or compact list
- **Roster management** - Players grouped by position with IR/TAXI/BYE status
- **Draft picks** - Future picks tracking (own + acquired) for dynasty
- **Standings** - Your position across all leagues with expandable full rankings
- **Trade history** - Recent trades in your dynasty leagues
- **Season history** - Championships and placements from 2020-2026
- **Bye week calendar** - See which players are on bye each week
- **Player exposure** - Position totals + most owned players across leagues
- **Age tracking** - Player ages and roster average age with color coding
- **Multi-language** - PT-BR, English, Spanish
- **Dark/Light theme** - Auto-detects system preference
- **PWA ready** - Install as app, works offline

### Quick Start
1. Open `index.html` in your browser
2. Enter your Sleeper username
3. Done!

### Tech
- Pure HTML/CSS/JavaScript (zero dependencies)
- Sleeper API only
- localStorage for cache and preferences
- Single file, ~50KB

### Deploy
Drop `index.html` on GitHub Pages, Netlify, or any static host.
